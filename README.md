# tic-tac-toe
This project is a simple implementation of the classic Tic Tac Toe game using HTML, CSS, and JavaScript. It is designed for two players who take turns marking spaces in a 3√ó3 grid with X and O. The first player to align three marks horizontally, vertically, or diagonally wins the game. If all spaces are filled without a winner, the match ends in a draw.
